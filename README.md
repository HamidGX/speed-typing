# speed-typing

Application designed to measure your typing speed and accuracy in real-time. It analyzes your performance and provides detailed results such as errors, accuracy, and characters typed. `Errors, Accuracy, and Characters typed.`

## Requirements

1. Node: Version 20.10.0 was used in this case.
2. Package Manager: You have the option to choose between NPM, Yarn, or other alternatives.

## Installation

```bash
yarn install
yarn dev
```

## Build

```bash
yarn build
```
